AI Email Assistant

A semantic search engine that finds emails by understanding meaning, not just keywords.

Built with Sentence Transformers, FAISS, & GitHub Actions

This demo searches a sample of 100+ real business emails.

How This Works

1. Your query is sent securely to this project's GitHub repository.

2. A GitHub Action triggers, running the Python ML model to find the most semantically similar emails.

3. The results are fetched and displayed here. Note: The initial search takes ~30 seconds as the Action starts.
